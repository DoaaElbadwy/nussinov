nussinov
========

Python implementation of Nussinov algorithm for RNA folding

Instalation
===========

```
git clone git@github.com:yakxxx/nussinov.git nussinov
sudo apt-get install virtualenv
virtualenv env
source env/bin/activate
pip install -r requirements.txt
```

Tests
=====

```
source env/bin/activate
python tests.py
```

Coverage
========

```
source env/bin/activate
coverage report
```
